Step-by-Step PC Build Instructions

Step-by-Step Build Guide

Step 1 - Instal CPU

Open the motherboard socket, the CPU has a triangle marker on it and align it with the motherboard socket.

Step 2 - Install RAM

Open the RAM slot clips on the motherboard,

Step 3 - Install SSD

Slide the SSD into the slot push down flat, secure it with a screw./p>

Step 4 - Install CPU Cooler

Apply thermal paste, mount the cooler over the cpu, secure with the brackets and screws.

Step 5 - Mount Motherboard on Case

Install by aligning the motherboard to the screws on the case.

Step 6 - Install Power Supply (PSU)

Place the psu having the fans going outward from the case, secure with screws.

Step 7 - Install GPU

Insert GPU where the motherboard tells you and clip it dow, until you here a noise.

Step 8 - Connect Case Cables

Connect the following cables to the motherboard there would be a guide.

Step 9 - First Power On

Click on the power button located on the pc case for first boot.

Step 10 - Install Windows and Drivers

Use a driver to install windows and drivers activate BIOS.
